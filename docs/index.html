Jabo Micro Game Engine is a micro game library originally developed for a game jam called JS13kgames that’s why Jabo Micro Game Engine is aiming for small file size yet powerful features. JS13KGames is a jam with a unique constraint on developing a game. In this jam you need to develop a game that has a maximum file size of 13kilobytes and your game must be written in JavaScript. Jabo Micro Game Engine provides many features a game must have such as animations, scene management, game object class, music and sound, and many more written in pure JavaScript with a tiny file size giving you a room for your game’s logics and graphics.
